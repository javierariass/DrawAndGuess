#--------Instrucciones de Juego---------------------


Fase Interpolar
1- Hacer click en el boton de "Bezier" para cambiar el metodo de interpolacion a Bezier
2- Hacer click en el boton de "Polinomio" para cambiar el metodo de interpolacion a Polinomio

Fase Dibujo
1- Hacer click izquierdo para colocar puntos y realizar el dibujo
2- Hacer click derecho para quitar el ultimo punto

Fase Adivinar
1- Teclea la palabra que creas segun el dibujo
2- Teclea backspace para borrar ultima letra
3- Revisa para comprobar tu respuesta
